SIP response construction (from RESEARCH.md Pattern 2) — key fields required in every response:
- Copy Via, From, To, Call-ID, CSeq verbatim from INVITE
- Add `tag=` to To header starting with 180 Ringing (generate once per call, reuse in 200 OK)
- 200 OK requires Contact header and Content-Type: application/sdp + Content-Length + SDP body
- 503/488 responses need only Content-Length: 0

SIP BYE construction (from RESEARCH.md Pattern 10) — dialog state needed:
- remoteTarget: Contact URI from INVITE Contact header
- fromUri: our SIP URI (sip:{SIP_USER}@{SIP_DOMAIN})
- fromTag: our local tag (the one we added to 180 Ringing)
- toUri: caller URI from INVITE From header
- toTag: caller's tag from INVITE From: ...;tag=xxx
- callId: from INVITE Call-ID
- cseq: start at 1, increment per BYE

ID format (CONTEXT.md locked decision):
- callSid: 'CA' + crypto.randomBytes(16).toString('hex')  (32 hex chars)
- streamSid: 'MZ' + crypto.randomBytes(16).toString('hex')  (32 hex chars)

   ```typescript
   function extractHeader(raw: string, name: string): string {

     return m ? m[1].trim() : '';
   ```

   ```typescript
   function buildResponse(p: { status, reason, via, from, to, callId, cseq, sdpBody?, localIp?, sipUser?, sipDomain?, localSipPort? }): string {
     const lines = [
       `SIP/2.0 ${p.status} ${p.reason}`,
       `Via: ${p.via}`,
       `From: ${p.from}`,
       `To: ${p.to}`,
       `Call-ID: ${p.callId}`,
       `CSeq: ${p.cseq}`,
     ];
     if (p.status === 200 && p.sdpBody) {
       lines.push(`Contact: <sip:${p.sipUser}@${p.localIp}:${p.localSipPort}>`);
       lines.push('Content-Type: application/sdp');
       lines.push(`Content-Length: ${Buffer.byteLength(p.sdpBody)}`);
       lines.push('');
       lines.push(p.sdpBody);
     } else {
       lines.push('Content-Length: 0', '');
     }
     return lines.join('\r\n') + '\r\n';
   }
   ```

**Private `terminateSession(session: CallSession, reason: string)`:**
```typescript
private terminateSession(session: CallSession, reason: string): void {
  if (!this.sessions.has(session.callId)) return; // idempotent
  this.sessions.delete(session.callId);
  session.log.info({ event: 'call_ended', reason }, 'Call terminated');
  session.ws.stop();      // sends stop event then closes WS
  session.rtp.dispose();  // closes dgram socket
}
```

**`terminateAll(): Promise<void>`:**
```typescript
async terminateAll(): Promise<void> {
  const sessions = [...this.sessions.values()];
  this.sessions.clear();
  await Promise.all(sessions.map(async (session) => {
    // Send SIP BYE to caller
    const bye = buildBye({ ... using session dialog state ... });
    // Parse remoteTarget to extract host and port
    const [byeHost, byePort] = parseContactTarget(session.remoteTarget);
    this.sipHandle.sendRaw(Buffer.from(bye), byePort, byeHost);
    session.log.info({ event: 'call_terminated_by_shutdown' }, 'Terminating call for shutdown');
    session.ws.stop();
    session.rtp.dispose();
  }));
}
```

`parseContactTarget(contactUri: string): [string, number]` — extract host and port from `sip:user@host:port` or `sip:host:port`:
```typescript
function parseContactTarget(uri: string): [string, number] {
  const m = uri.match(/[@:]([a-zA-Z0-9._-]+):(\d+)/);
  if (m) return [m[1], parseInt(m[2], 10)];
  const host = uri.match(/[@:]([a-zA-Z0-9._-]+)/)?.[1] ?? '127.0.0.1';
  return [host, 5060];
}
```
