<decisions_made>

- `SDP_CONTACT_IP` made optional — auto-detects via `net.Dial("udp","8.8.8.8:80")` if not set
- REGISTER From/To must be pre-set as `sip:user@domain` before calling `Do()` — sipgo's `clientRequestBuildReq` uses `c.UserAgent.name` (not the SIP user) for From.User
- REGISTER Contact header must be manually added as `<sip:user@SDPContactIP:5060>` — `ClientRequestRegisterBuild` does NOT add Contact
- UNREGISTER uses `Contact: *` (RFC 3261 §10.2.2 — deregister all bindings)
- `defer dlg.Close()` must NOT be in onInvite — it removes dialog from cache before ACK/BYE. `ReadBye` already calls `Close()` internally via its own defer.
</decisions_made>
