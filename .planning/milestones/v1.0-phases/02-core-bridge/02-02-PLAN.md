connected event:
```json
{"event":"connected","protocol":"Call","version":"1.0.0"}
```

start event:
```json
{
  "event": "start",
  "sequenceNumber": "1",
  "start": {
    "streamSid": "MZ...",
    "accountSid": "",
    "callSid": "CA...",
    "tracks": ["inbound","outbound"],
    "customParameters": {"From":"...","To":"...","sipCallId":"..."},
    "mediaFormat": {"encoding":"audio/x-mulaw","sampleRate":8000,"channels":1}
  },
  "streamSid": "MZ..."
}
```

media event (outbound — from us to backend):
```json
{
  "event": "media",
  "sequenceNumber": "N",
  "media": {"track":"inbound","chunk":"N","timestamp":"N","payload":"<base64>"},
  "streamSid": "MZ..."
}
```

stop event:
```json
{
  "event": "stop",
  "sequenceNumber": "N",
  "stop": {"accountSid":"","callSid":"CA..."},
  "streamSid": "MZ..."
}
```

dtmf event:
```json
{
  "event": "dtmf",
  "streamSid": "MZ...",
  "sequenceNumber": "N",

}
```

media event (inbound — from backend to us, what we receive):
```json
{"event":"media","media":{"payload":"<base64 mulaw>"}}
```

```typescript
export interface WsClient {
  /** Forward mulaw payload Buffer from RTP as inbound media event */
  sendAudio(payload: Buffer): void;
  /** Forward DTMF digit string as dtmf event */
  sendDtmf(digit: string): void;
  /** Send stop event, then close the WebSocket */
  stop(): void;
  /** Register handler for outbound audio payload from WS backend (decode base64 → Buffer) */
  onAudio(handler: (payload: Buffer) => void): void;
  /** Register handler for WS close/error — called when WS disconnects for any reason */
  onDisconnect(handler: () => void): void;
}
```

**Export `createWsClient(url: string, params: WsCallParams, log: Logger): Promise<WsClient>`:**

Connect with 2-second timeout (RESEARCH.md Pattern 8 + CONTEXT.md locked decision):
```typescript
const ws = new WebSocket(url);
const timer = setTimeout(() => {
  ws.terminate();
  reject(new Error(`WS connect timeout after 2000ms to ${url}`));
}, 2000);

ws.once('open', () => {
  clearTimeout(timer);
  // send connected + start
  resolve(buildClient());
});

ws.once('error', (err) => {
  clearTimeout(timer);
  reject(err);
});
```

After `open` fires, IMMEDIATELY send (in sequence, synchronous):
1. `ws.send(JSON.stringify({ event: 'connected', protocol: 'Call', version: '1.0.0' }))`
2. `ws.send(JSON.stringify({ event: 'start', sequenceNumber: '1', start: { streamSid, accountSid: '', callSid, tracks: ['inbound','outbound'], customParameters: { From: from, To: to, sipCallId }, mediaFormat: { encoding: 'audio/x-mulaw', sampleRate: 8000, channels: 1 } }, streamSid }))`

Per-session state for sequencing:
- `let seq = 2` (1 was used by start event)
- `let chunk = 0`
- `let rtpTimestamp = 0`

`sendAudio(payload: Buffer)`:
- If `ws.readyState !== WebSocket.OPEN`: log.warn and return (drop packet — per CONTEXT.md locked decision)
- `ws.send(JSON.stringify({ event:'media', sequenceNumber:String(seq++), media:{ track:'inbound', chunk:String(chunk++), timestamp:String(rtpTimestamp), payload: payload.toString('base64') }, streamSid }))`
- `rtpTimestamp += 160` (20ms at 8kHz)

`sendDtmf(digit: string)`:
- If not OPEN: log.warn and return
- `ws.send(JSON.stringify({ event:'dtmf', streamSid, sequenceNumber:String(seq++), dtmf:{ track:'inbound_track', digit } }))`

`stop()`:
- If OPEN: send stop event: `ws.send(JSON.stringify({ event:'stop', sequenceNumber:String(seq++), stop:{ accountSid:'', callSid }, streamSid }))`
- Then `ws.close()` (graceful close frame)
- If not OPEN: `ws.terminate()` (skip stop event, just terminate)

`onAudio(handler)`:
- Register `ws.on('message', (data) => { ... })`
- Parse JSON, check `msg.event === 'media'`
- `handler(Buffer.from(msg.media.payload, 'base64'))`
- Ignore non-media events (mark, clear, etc.)

`onDisconnect(handler)`:
- `ws.once('close', handler)`
- `ws.once('error', (err) => { log.error({ err, event: 'ws_error' }, 'WS error'); handler(); })`

Anti-patterns to avoid (from RESEARCH.md):
- Do NOT send 200 OK before WS is connected — this module resolves AFTER `open` fires and after connected+start are sent
- Do NOT buffer audio while WS is reconnecting — sendAudio drops silently if not OPEN
- Do NOT share this WsClient between calls — createWsClient creates a fresh WebSocket per call (CON-01)
