Was ist das?

Eine Brücke zwischen deiner App und Deutschland.

Stell dir vor, du hast eine App gebaut, die telefonieren kann. Die App spricht eine bestimmte Sprache — die Sprache von Twilio. Das Problem: Twilio ist eine amerikanische Firma. Deine Telefongespräche gehen durch Server in den USA.

Die sipgate SIP Stream Bridge ist die Lösung. Sie übersetzt die Twilio-Sprache deiner App in echte deutsche Telefonie — über sipgate, einem Anbieter aus Düsseldorf.

Deine App merkt nichts davon. Sie denkt immer noch, sie redet mit Twilio. Aber in Wirklichkeit telefoniert sie über Deutschland. Kein Code anfassen. Kein Risiko. Volle Kontrolle.

Warum sipgate?

Vier gute Gründe,
den Wechsel zu machen.

🇩🇪
Daten bleiben in Deutschland
Twilio ist US-amerikanisch. Deine Telefongespräche verlassen Europa und landen auf amerikanischen Servern, die amerikanischem Recht unterliegen. Mit sipgate bleibt alles in Deutschland — kein Umweg über den Atlantik, kein Datenschutzproblem.
Serverstandort: Deutschland
🔒
DSGVO — kein Kopfschmerz mehr
Kein Auftragsverarbeitungsvertrag mit einem US-Unternehmen. Keine Standardvertragsklauseln. Kein Anwalt, der das prüfen muss. Deutsches Unternehmen, deutsches Recht. DSGVO-Compliance ohne Kopfrechnen. Fertig.
💶
Günstigere Telefonie, direkt
Echte deutsche Festnetz- und Mobilfunknummern zu fairen Preisen — direkt, ohne Umweg über einen amerikanischen Vermittler. Die Bridge selbst ist komplett kostenlos.
🤝
Support, der dich versteht
sipgate ist seit über 20 Jahren in Deutschland. Deutschsprachiger Support. Deutsche Mentalität. Wenn etwas nicht klappt, rufst du einfach an — auf Deutsch, ohne Ticketsystem.

FAQ

Häufige Fragen.

Muss ich meinen Code umschreiben?
+
Nein. Die Bridge spricht exakt dieselbe Sprache wie Twilio Media Streams. Dein Backend merkt keinen Unterschied — du musst keinen einzigen Code anfassen.
Brauche ich einen sipgate Account?
+
Ja — du brauchst einen sipgate trunking-Anschluss. Den bekommst du unter sipgatetrunking.de. Die Bridge selbst ist kostenlos und open source.
Ist das wirklich DSGVO-konform?
+
sipgate ist ein deutsches Unternehmen mit deutschen Servern. Deine Telefondaten verlassen Deutschland nicht. Das ist ein erheblicher Vorteil gegenüber US-Anbietern wie Twilio, bei denen du Standardvertragsklauseln und einen Auftragsverarbeitungsvertrag brauchst.
Was kostet die Bridge?
+
Die Bridge ist komplett kostenlos und open source. Du hostest sie selbst — aber keine Sorge: Das Docker-Image ist nur ~10 MB groß und läuft problemlos auf dem kleinsten Cloud-Server (1 vCPU, 512 MB RAM). Du zahlst nur für deinen sipgate-Anschluss.
Wie viele Anrufe kann die Bridge gleichzeitig verarbeiten?
+
Standardmäßig bis zu 100 parallele Gespräche — konfigurierbar nach oben. Läuft stabil über Wochen ohne Neustart.
Läuft das auch ohne Linux?
+
Für Produktion empfehlen wir Linux mit --network host. Lokal auf macOS oder Windows funktioniert Docker Desktop ebenfalls.
